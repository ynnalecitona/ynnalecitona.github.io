Mini Chess Player

Implemented a program in Python that plays mini chess that includes two functions, one is a move generator that returns a list of possible new boards that can result from the current board in one move depending on color.

It takes in consideration legal moves for both pawns.

The second function evaluates each of the boards and available moves and then returns the best one.
